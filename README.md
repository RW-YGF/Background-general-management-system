###  项目简介：
这是一个基于VUE3的小型的后台通用系统前端，利用Element Plus组件库搭建项目页面。
首页利用echarts实现折线图、柱状图和饼状图。
用户管理页使用axios交互数据，使用mockjs模拟数据。
